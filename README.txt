Wedding Invitation
====

## Description

結婚式＆披露宴用のWebページ。
とりあえずメールが送れるようになったらそのまま使用を検討する。

## Tools
FREEHTML5.COのテンプレを流用している。
netlifyでフォームを使うためにホスティングしている。
独自ドメインについては未定。

## Form
GoogleFormを利用している。

## Compile

SCSSを利用する。
CSSへのコンパイルにはIDE等でコンパイルし、SCSSもCSSも両方共にコミットさせる。
